# Humanizador de Textos

Una herramienta web sencilla para simplificar el vocabulario de textos complejos o generados por IA, convirtiéndolos en lenguaje más natural y humano.

## 🚀 Funcionalidades
- Detección de "IA-ismos" (palabras excesivamente formales).
- Reemplazo automático por sinónimos comunes.
- Resaltado visual de los cambios realizados.
- Opción de copiar el resultado con un solo clic.

## 🛠️ Tecnologías utilizadas
- HTML5
- Tailwind CSS (vía CDN)
- JavaScript Vanilla

## 📂 Instalación
1. Clona el repositorio.
2. Abre `Humanizador.html` en cualquier navegador moderno.
